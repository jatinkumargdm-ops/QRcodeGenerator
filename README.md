# QR Code Generator (Python)

A mini Python project that generates a QR code from a text or URL in just a few lines of code.

## Features

* Generate QR codes instantly
* Supports text, URLs, and other string data
* Beginner-friendly and lightweight

## Requirements

* Python 3.x
* `qrcode` library

Install the required package:

```bash
pip install qrcode[pil]
```

## Usage

1. Open the Python file.
2. Replace the sample text or URL with your own and image name(here Github.png)with your own.
3. Run the script:

```bash
python QRcode.py
```

A QR code image will be generated and saved in the project directory.

## Example

Input:

```
https://github.com/your-username
```

Output:

```
Github.png
```
